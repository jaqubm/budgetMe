## Development

### Backend

1. Navigate to the project root
2. Start the backend server:
   ```bash
   npm run backend:dev
   # or
   uv run server
   ```
3. API Documentation:
   - Swagger UI: http://localhost:8000/docs
   - ReDoc: http://localhost:8000/redoc

### Frontend

1. Navigate to the project root
2. Start the frontend dev server:
   ```bash
   npm run frontend:dev
   ```
3. Open http://localhost:5173
